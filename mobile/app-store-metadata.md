## Data Safety & Privacy

### Apple App Store - App Privacy

**Data Used to Track You:**
- None (we do not track users across apps/websites for advertising)

**Data Linked to You:**
- Contact Info: Email address, Name
- User Content: Photos (for profile/exam questions), Videos (proctoring recordings)
- Identifiers: User ID
- Usage Data: Product interaction (analytics)
- Diagnostics: Crash logs, Performance data

**Data Not Linked to You:**
- Diagnostics: Anonymous usage analytics

**Privacy Practices:**
- Data is encrypted in transit using HTTPS/TLS 1.3
- Data is encrypted at rest using AES-256
- Users can request data deletion
