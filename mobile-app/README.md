# ProficiencyAI Mobile App

## Quick Start

To get the QR code for iPhone testing:

```bash
cd mobile-app
npx expo@latest start --tunnel
```

## App Features

- Login with test@example.com
- View real quiz data from ProficiencyAI backend
- Mobile-optimized quiz interface
- Connect to live Replit server

## Backend Connection

## Testing

1. Install Expo Go on iPhone
2. Run the command above
3. Scan QR code that appears
4. App loads with real data
