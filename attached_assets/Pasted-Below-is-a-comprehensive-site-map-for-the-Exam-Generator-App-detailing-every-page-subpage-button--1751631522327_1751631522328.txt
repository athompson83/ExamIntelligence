Below is a comprehensive site map for the Exam Generator App, detailing every page, subpage, button, link, table, menu, and form required to support all features for both teacher/administrator (primarily on PC) and test taker (PC and native mobile) experiences.

1. Public & Authentication
Landing Page / Home

Components:

Branding (logo, tagline)

Overview of features

“Learn More” links

Footer links: Help/FAQ, Contact, Privacy Policy, Terms of Service

Authentication Pages:

Login Page:

Form fields: Email/Username, Password

Buttons/Links: “Login”, “Forgot Password?”, “Sign Up” (if self-registration is allowed)

Registration Page (if applicable):

Form fields: Name, Email, Password, Role selection (if self-service for teachers is allowed)

Button: “Register”

Password Reset Page:

Email input

Button: “Send Reset Link”

2. Teacher/Administrator Interface
A. Global Layout (Common for All Teacher Pages)
Top Bar:

Logo (clickable – returns to Dashboard)

Notifications icon (dropdown list with alerts: validation issues, exam alerts, etc.)

User Profile Menu (Profile, Settings, Logout)

Sidebar Menu (Persistent Navigation):

Dashboard

Testbanks

Quiz Builder

Exam Hosting

Analytics

Resources

Settings

(Optional) Help/Documentation

B. Dashboard
Dashboard Main Page:

Overview Widgets/Cards:

Upcoming Exams (count, mini table or graph)

Recent Testbank Updates (list of latest modified testbanks/questions)

Live Exam Alerts (if any active sessions)

Quick Actions:

“Create New Testbank” button

“Create New Quiz” button

“Revalidate Questions” button

Activity Feed: Table or list showing recent revisions, AI validations, peer review comments

Links:

Drill-down links to detailed pages (e.g., click “Upcoming Exams” to go to the Exam Hosting section)

C. Testbank Management
Testbank List Page:

Elements:

Search Bar / Filter Options (tags, Bloom’s levels, learning objectives, date range)

List/Grid/Table of Testbanks with columns/cards displaying:

Title

Description snippet

Creation Date

Last Revalidated Date

Number of Questions

Buttons on Each Item:

“Edit” (link to Testbank Detail/Editor)

“Delete”

“Revalidate” (triggers AI-powered check)

Global Action: “Create New Testbank” button

Testbank Detail / Editor Page:

Header:

Testbank Title, Description, Metadata

“Edit” and “Delete” buttons

“Revalidate” button

Main Content:

Questions Table/List:

Columns: Question Text, Type, Difficulty, Last Validated, Tags

Action Buttons per Row: “Edit Question”, “Delete”, “View Revisions”, “View Peer Reviews”

Action: “Add New Question” button

D. Question Editor & Detail
Question Editor Page:

Form Fields:

Question Prompt: Rich text editor

Question Type: Dropdown (MC, MR, CR, Hot Spot, etc.)

Answer Options Section (if applicable):

Repeating group for Answer Options with fields:

Answer Text

“Is Correct?” toggle

Media Attachment (if needed)

“Add New Option” button

“Remove” icon/button for each option

Metadata Panel:

Input fields/tags for: Bloom’s taxonomy level, Learning Objectives, Custom Categories

AI Validation Panel:

Display area for AI-generated feedback, suggestions, and difficulty recalibration

“Revalidate” button

Tabs/Subsections:

Revision History Tab: Table with columns (Modified By, Date, Change Description, “View Previous Version” link)

Peer Review Tab: List of reviews with Reviewer, Rating, Comments

Footer/Action Buttons:

“Save”, “Preview”, “Publish”, “Cancel”

E. Quiz Builder / Creation
Quiz List Page:

Table/List:

Columns: Quiz Title, Description, Creation Date, Scheduled Times, Status

Action Buttons per Quiz: “Edit”, “Delete”, “View Results”

Global Action: “Create New Quiz” button

Quiz Creation / Editor Page:

Form Fields:

Quiz Title, Description, Time Limit, Multiple Attempts (toggle)

Security Options:

Password Protection (toggle + input for password)

IP Locking (toggle)

Adaptive Testing Settings:

Section for setting adaptive rules (if correct then show harder question, etc.)

Dual-Pane Interface:

Left Pane:

List of Available Questions (from Testbanks)

Search and Filter Options

Checkboxes or “Add” buttons beside each question

Right Pane:

Current Quiz Assembly: Ordered list/table of selected questions

Drag-and-drop or “Move Up/Down” buttons

“Remove” button for each question

Randomization Options:

Toggle for Shuffling Questions

Toggle for Randomizing Answer Options

Footer/Action Buttons:

“Save”, “Preview”, “Publish”, “Cancel”

F. Exam Hosting & Monitoring
Exam Scheduler Page:

Form Fields:

Exam Title, Description

Date/Time Pickers for Start and End Times

Security Options (as in Quiz Builder)

Calendar/Timeline View:

Visual scheduling interface showing upcoming exams

Action Buttons:

“Schedule Exam”, “Edit Exam”, “Delete Exam”

Live Exam Monitoring Dashboard:

Table/List of Active Exams:

Columns: Exam Name, Teacher, Start Time, Active Participants, Status

Each exam row is clickable to open Exam Details

Exam Details Page:

Student List Table:

Columns: Student Name, Status (In Progress, Completed), Time Taken

Proctoring Alerts column (flag icon, link to details)

Proctoring Log Section:

Table listing events with columns: Timestamp, Event Description, Screenshot (if available), Flag Status

G. Analytics & Reporting
Analytics Dashboard Page:

Data Visualizations:

Charts and Graphs (e.g., pass/fail rates, average time per question, performance trends)

Drill-down clickable elements to access detailed reports

Filters:

Dropdowns/Date Pickers for filtering by Testbank, Quiz, Date Range, Question Type

Tables:

Summary tables of detailed question-level analytics and exam performance

H. Resources & AI Content
Resources Page:

List/Grid:

Cards or rows for each resource (Study Guides, Lecture Notes, Improvement Plans)

Preview snippet, Resource Type indicator

Action Buttons:

“View Resource”, “Edit Resource”, “Delete Resource”, “Generate New Resource” (triggers AI generation)

I. Settings
Profile Settings Page:

Form Fields:

Name, Email, Profile Picture, Password (with change option)

Buttons: “Save”, “Cancel”

App Settings Page:

Options:

Language, Notification Preferences

API/Integration Settings (for AI, Proctoring, LMS integrations)

Buttons: “Save Settings”, “Reset”

General Site Settings:

Links to Legal Pages (Privacy Policy, Terms of Service)

J. Notifications / Alerts
Notifications Dropdown (Top Bar):

List of recent notifications with timestamps and short messages

Link: “View All Notifications” (opens a dedicated Notifications Page)

Notifications Page:

Table/List:

Columns: Notification Type, Message, Date/Time, Read Status

Action: Mark as Read, Delete Notification
